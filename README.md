## LRU Cache
A cache with limited capacity that removes the least recently used item when full.
Uses OrderedDict to track usage order - reading an item moves it to the end.
## Tokenizer (Word Frequency Counter)
A text tokenizer that splits text into words and counts how often each word appears.
Built as a reusable function.
## Two Sum
Finds two numbers in a list that add up to target, return their positions.
Uses a dictionary for last lookup of seen numbers.
## Valid Parentheses
Checks if brackets in a string are correctly opened and closed in order.
Uses a stack (list) - push opening brackets, pop and match on closing.
## Reverse Linked List
Reverses a singly linked list by flipping each node's pointer backwards.
Uses a Node class and three pointers (prev,current, next)
